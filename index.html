EUR Installation Guidelines:

DaC V5 is needed for the installation of EUR

  1. Download the most recent EUR version
  2. Unzip the file to any location.
  3. Drag and drop the contents into DAC folder and overwrite it
  4. Rename your Divide_and_Conquer folder to Divide_and_Conquer_EUR
    very important. Otherwise your game wont launch!
  5. Run Large Address Aware on the Medieval2.exe
  6. Run cleaner.bat
  7. Launch game from M2TWEOP GUI
